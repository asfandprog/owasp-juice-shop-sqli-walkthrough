# OWASP Juice Shop SQL Injection Walkthrough

## Overview

This repository documents my learning experience with SQL Injection concepts using OWASP Juice Shop, a deliberately vulnerable application created for security education.

## Objectives

* Understand what SQL Injection is
* Learn how insecure input handling can affect applications
* Explore web application security concepts
* Practice documenting security findings

## Lab Environment

* OWASP Juice Shop
* Kali Linux
* Google Chrome
* Local Lab Environment

## What I Learned

### SQL Injection

SQL Injection is a web security vulnerability that occurs when an application improperly handles user input before interacting with a database.

### Potential Risks

* Unauthorized access
* Data exposure
* Data modification
* Application compromise

### Security Best Practices

* Use parameterized queries
* Validate user input
* Apply least-privilege access controls
* Perform regular security testing
